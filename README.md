# Brnd Monk

Brnd Monk is a modern mobile application built with React Native and Expo. It features a sleek dark UI and integrates with Firebase for backend services.

## Features

- **Modern UI**: Dark-themed user interface with smooth animations and gradients.
- **Firebase Integration**: Robust backend support for data and authentication.
- **Cross-Platform**: Designed to run seamlessly on both iOS and Android.
- **Image Handling**: Integrated image picker for content creation.
- **Export & Sharing**: Support for printing and sharing content directly from the app.

## Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) with [Expo](https://expo.dev/)
- **Navigation**: [React Navigation](https://reactnavigation.org/)
- **Backend**: [Firebase](https://firebase.google.com/)
- **Styling**: [React Native Reanimated](https://react-native-reanimated-docs.swmansion.com/) for fluid animations.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Expo CLI (`npm install -g expo-cli`)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/jnaneshshetty/brndmonk.git
   cd brndmonk
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Development

- `npm run android` - Start on Android emulator/device.
- `npm run ios` - Start on iOS simulator/device.
- `npm run web` - Start on web browser.
